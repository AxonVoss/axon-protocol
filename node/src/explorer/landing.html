The Protocol

How Proof of Agent Work works

Every AXON block requires two proofs — one mathematical, one cognitive.

01
Challenge Derived
A unique challenge is computed from the previous block hash, height, and miner address. No two challenges are ever the same.
02
Inference Runs
The challenge is passed to the canonical TinyLlama model. The model's output is hashed. This hash must meet the PoAW difficulty target.
03
PoW Solved
Standard hashcash Proof of Work also runs in parallel. Both the PoW nonce and PoAW nonce must be valid for the block to be accepted.
04
Block Sealed
The inference hash is embedded permanently in the block header — cryptographic proof that an AI agent did real cognitive work to mine this block.
Why AXON

A different kind of consensus

Neither PoW nor PoS has anything to do with intelligence. AXON changes that.

Bitcoin
Proof of Work
Rewards whoever has the most hardware. Mining is an energy arms race with no meaningful output beyond the block itself.
Ethereum
Proof of Stake
Rewards whoever already has the most tokens. Replaces the energy arms race with a wealth arms race.
AXON ⚡
Proof of Agent Work
Rewards agents for running inference. CPU-mineable, fair launch, no pre-mine. Intelligence is the scarce resource.
Get Started

Start mining in 5 steps

Any machine with a CPU can participate. No GPU, no ASIC, no specialist hardware required.

bash
# 1. Clone and install
git clone https://github.com/AxonVoss/axon-protocol
cd axon-protocol/node && npm install

# 2. Download the canonical model (one-time, ~639 MB)
npx ts-node src/cli.ts setup-inference

# 3. Generate your wallet — write down your 24 words
npx ts-node src/cli.ts new

# 4. Check your balance anytime
npx ts-node src/cli.ts balance <your-address>

# 5. Start mining
PEERS=seed1.axonprotocol.net:8333 \
npx ts-node src/cli.ts mine 999999 <your-address>
